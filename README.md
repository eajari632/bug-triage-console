# Bug Triage Console

Founder’s Office Case Study Submission by Esther Ajari

## Live Prototype
https://eajari632.github.io/bug-triage-console/

## Purpose
A lightweight, auditable bug-triage workflow for customer-reported voice-agent issues.

## Features
- Rules-based classification
- Severity scoring
- Compliance flagging
- Human override workflow
- CSV export
- Audit trail

## Run Locally
Download or clone this repository, then open `index.html` in Chrome, Edge, or Firefox.

## Notes
This prototype was built using Claude Code as a vibe-coding tool and refined through manual QA of all 15 test reports.
